# Introduction to HTML Forms

HTML forms are used to get data from the users in the applications or websites. A form can be used to log in a user, sign up a user or get any other information.

## HTML Form elements

1. Input (text, number, password, radio (same name), checkbox)
2. Textarea
3. Select (single and multiple)
4. Datalist
5. Label
6. Button

## HTML Form Extra Things

1. Validations (required, min, max)
2. Autofocus
3. Autocomplete (current-password, new-password, username)
4. Attributes (checked, disabled)

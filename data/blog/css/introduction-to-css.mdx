# Introduction to CSS

CSS stands for Cascading Style Sheets. It is used to style html elements in a document.

## Main topics

1. CSS Box Model
2. CSS units (Absolute vs. relative)
3. CSS units (px, pt, in, rem, em, vh, ch, %)
4. HTML Landmark elements
5. How to build containers
6. CSS Flex
7. Absolute and Relative positioning

### CSS Flex Properties of Parent

- display
- flex-direction
- flex-wrap
- justify-content
- align-items
- align-content (like justify-content but for multi-line flex-containers with wrapping)
- gap, row-gap, column-gap

### CSS Flex Properties of Children

- order
- flex-grow
- flex-shrink
- flex-basis (Default size of an element before extra space is distributed)
- align-self
